## ◆ まず前提：なぜこんなにいろいろ聞くのか

My Focus は、クリエイターさんが写真や動画を投稿して、ユーザーが買う仕組みです。

このタイプのサービスは、**運営している会社（＝91&Co.さん）が「アップロードされた中身」について責任を持つ**ルールに法律でなっています。具体的には：

- もし18歳未満のコンテンツが投稿されてしまったら→ 91&Co.さんが警察沙汰
- 著作権侵害（他人の写真を勝手にアップ）の通報がきたら→ 91&Co.さんが対応する義務
- 違反コンテンツを放置すると→ 91&Co.さんが共犯扱いになることもある

なので「**起きてから対応する**」ではなく「**起きないように設計する**」必要があります。
そのために以下を決めさせてください。

## 1. 投稿される写真・動画を、公開前にチェックするかどうか

### なぜ聞くか

クリエイターが投稿した写真や動画が、すぐサイトに公開されるか、それとも一度誰かが確認してから公開されるかを決める必要があります。

「全部後で見ます」だと、違反コンテンツが数時間〜数日サイトに出てしまい、その間に問題が起きた場合、91&Co.さんの責任になります。

### 質問

**Q1-1. 投稿されたコンテンツをどうやってチェックしますか？（4択）**

- [ ] **A. 人が全部目視チェック**
  - 1件1件、運営スタッフが見てから公開ボタンを押す
  - 一番安全だが、運営スタッフの人件費が一番かかる
- [ ] **B. AIに任せて自動公開**
  - AIが「これは違反っぽい」と判定したものだけ止める
  - 早く・安いが、AIをすり抜けるリスクあり
- [ ] **C. AIで一次チェック → 怪しいものだけ人が確認**（💡植松おすすめ）
  - 99%はAIが安全と判定して即公開、1%だけ人がチェック
  - コスト・スピード・安全のバランスがベスト
- [ ] **D. 全部すぐ公開して、通報があった時だけ対応**
  - 一番楽だが、法的リスクが一番高い

**Q1-2. チェック作業を実際にやるのは誰ですか？**

- [ ] 91&Co.内のスタッフがやる
- [ ] 外部に委託する（業者紹介できます）
- [ ] まだ決まっていない／植松と相談したい

**Q1-3. 違反コンテンツの通報が入った時、どれくらいの時間で対応すると約束しますか？**

> 補足：規約に書く時間です。「24時間以内に削除します」と書くと、それを守る義務が生じます。

- [ ] 24時間以内に対応（一般的）
- [ ] 48時間以内に対応（少し余裕あり）
- [ ] 営業日内に対応（休日除く、運営しやすい）
- [ ] まだ決められない

## 2. AIチェックを入れる場合、月いくらまでなら出せるか

### なぜ聞くか

AIで自動チェックする場合、月額のランニング費用がかかります。
精度が高いほど高いです。月の投稿数1,000件で試算すると：

| サービス | 月額目安 | 精度 | 備考 |
|---|---|---|---|
| AWS Rekognition | 3,000円〜8,000円 | ふつう | 安いがアダルト判定はやや弱い |
| Sightengine | 5,000円〜12,000円 | やや高 | 月額固定でわかりやすい |
| Hive Moderation | 15,000円〜25,000円 | 高い | アダルト特化、業界標準 |
| 人だけで対応 | 50,000円〜150,000円 | 最高 | 運営の手間がすごい |

### 質問

**Q2-1. AI審査の月額予算、上限はいくらまでOKですか？**

- [ ] 〜5,000円（AWSで様子見）
- [ ] 〜15,000円（Sightengine）
- [ ] 〜30,000円（Hiveでがっつり）💡植松おすすめ
- [ ] とりあえず安く始めて、必要に応じて切替えたい
